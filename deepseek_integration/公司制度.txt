# 新员工入职培训（员工手册篇）
## 第一章 人事管理制度
### 一、入职流程
#### 1. 入职准备与初期手续
- 提交入职资料：需提供身份证、学历证书、学位证书、专业技术证书、个人简历、离职证明、社保用电子版照片，同时填写并提交《员工入职登记表》《XXXX公司入职职业规划表》。所有资料需保证真实，若存在虚假，公司将予以辞退且不作任何经济补偿。
- 办理入职手续：完成资料提交后，办理相关入职手续。
- 录入指纹考勤：入职手续办理完毕后，录入指纹用于日常考勤。
- 提交体检报告：需在规定时间内提交体检报告。
- 签订试用期合同：完成上述步骤后，签订试用期合同。
- 进入试用期：签订试用期合同后，正式进入试用期，试用期为3个月。
- 办理社保五险：员工进入试用期后，公司为其办理社保五险。

#### 2. 入职培训
员工需参加公司组织的入职培训，了解公司相关制度、文化等内容。

#### 3. 试用期满与转正
- 提交转正申请：试用期内表现优秀的员工，可向部门主管申请提前转正；试用期正常结束前，员工需提交转正申请。
- 评估转正申请：部门对员工的转正申请进行评估。
- 转正员工面谈：评估通过后，与转正员工进行面谈。
- 签订劳动合同：面谈通过后，签订有效期为3年的劳动合同。

### 二、劳动合同
#### 1. 合同签订时间
员工入职一个月内，公司与员工签订劳动合同。

#### 2. 试用期规定
试用期为3个月，试用期内表现优秀者可向部门主管申请提前转正。

#### 3. 离职程序
- 试用期内离职：员工须提前三天提交离职申请。
- 正式员工离职：须提前一个月提出书面申请，方可办理离职手续。

### 三、考勤制度
#### 1. 工作时间
实行5天8小时工作制，具体时段如下：
- 上午：09:00-13:00（茶歇时间：11:15-11:30）
- 下午：14:00-18:00

#### 2. 考勤记录方式
采用指纹刷卡记录考勤。

#### 3. 考勤异常界定
- 漏打卡、无故不打卡均视为旷工。
- 严禁代打卡。
- 迟到/早退：晚于规定上班时间到岗且未超过30分钟（含），或早于规定下班时间离岗且未超过30分钟（含）。
- 擅离岗位/逾期未返岗：未办理请假手续、请假手续未获批即擅自离岗，或假期届满无续假仍未到岗。
- 严重迟到/早退：晚于规定上班时间或早于规定下班时间超过30分钟。
- 违法违规被拘留、逮捕。
- 骗取、涂改、伪造休息休假证明。

#### 4. 全勤奖励
全月无请假、迟到、早退、旷工情况的员工，当月奖励100元全勤奖金。

#### 5. 迟到早退处理
| 迟到或早退时长T | 处理办法 |
| ---- | ---- |
| 5≤T≤15分钟 | 扣款10元/次 |
| 15分钟<T≤30分钟 | 扣款20元/次 |
| T>30分钟 | 视作旷工半天处理 |
注：所罚款项将作为所属部门的活动经费；原则上公司允许员工善意迟到2次。

#### 6. 旷工处理
员工旷工按日工资标准的双倍扣罚，具体附加处理如下：
- 旷工半天：全公司内通报并警告一次。
- 旷工一天：全公司通报并记过一次。
- 连续旷工3天以上、月累计旷工5天或年累计旷工10天以上：公司除按上述标准扣罚考勤款外，将与其解除劳动合同，不予支付任何经济补偿金或赔偿金，并保留追索权益损失的权利。

### 四、请假制度
#### 1. 申请与销假
- 申请：至少提前一天提交书面《请假单》；请假期间工作需在假期开始前完成交接。
- 销假：请假结束后，需第一时间到人事处销假。

#### 2. 不同假期提前申请时间
- 请假1天以内：至少提前一个工作日向权限审批人提出。
- 请假1天以上：至少提前一个工作日向权限审批人提出。
- 连续休假5天：至少提前一个工作日向权限审批人提出。
- 婚假：至少提前十天提出。
- 产假：至少提前一个工作日向权限审批人提出（具体可根据实际情况调整）。
- 急性病、突发性疾病：允许当日口头请假，就诊后于三日内补交相关证明单据。

#### 3. 病假特别规定
- 未经批准休病假或未按规定提交相关证明材料的，均视为旷工。
- 采用虚假证明（含骗取、伪造、篡改、涂改病假证明）休病假的，视为违反规章制度，按旷工处理。
- 因违反治安管理规定或自杀、自残导致停工治疗的，不视为病假，按事假处理。
- 病假需提供正规医院出具的证明材料：诊断证明书及休假建议条、门诊病历、医药费收据、挂号证明、出院记录等原件，行政部留存复印件备案。
- 病假工资按北京市最低工资标准的80%发放；病假期间补贴按日扣除。
- 医疗期满后，因身体疾病导致出勤率低于80%的，视为不能从事工作。

### 五、休假制度
#### 1. 法定假
每年共11天法定休假日，具体安排如下：
- 元旦：1天（1月1日）
- 春节：7天（含调休）
- 清明节：3天
- 劳动节：3天（含调休）
- 端午节：3天
- 中秋节：7天（含调休）
- 国庆节：7天（含调休）
- 妇女节：全体女性员工放假半天，若遇周六、周日，不予另行补假。

#### 2. 婚假
- 适用条件：符合法定结婚年龄（男22周岁，女20周岁）；晚婚年龄（男25周岁，女23周岁）。
- 假期时长：婚假为带薪假。员工在公司工作期间登记结婚，可享受3天（自然天数）婚假；晚婚者另增加12天晚婚假；再婚者仅可享受法定婚假3天，不享受晚婚假。
- 休假要求：婚假包含休息日和法定休假日，原则上需在结婚登记后一年内一次性休完，确因工作需要无法一次性休完的，最多分两次使用，剩余假期不予补休；请休婚假须至少提前十天提出，凭结婚证办理休假手续，或返岗当日将结婚证原件、复印件交行政部核验备案（原件核验后交还）。

#### 3. 产假
- 前提条件：遵守国家计划生育政策（已婚、计划内生育）。
- 假期分类及时长：
  - 产前检查假（女）：怀孕后可遵医嘱定期产检，每次半天，一般不超过15次；初次请休需提供医院检查证明。
  - 生育产假（女）：基础90天（含产前15天）；初育晚育增加30天；晚育且领《独生子女父母光荣证》增加44天；剖宫产增加15天；多胞胎生育的，每多生育一名婴儿增加15天。
  - 产后陪护假（男）：7天。
- 特殊考勤：女员工怀孕期间可申请推迟半小时上班；怀孕7个月（含）以上，部门可根据工作需要每天安排一小时额外休息时间，计入劳动时间。
- 流产假期：
| 怀孕时长 | 流产假期 |
| ---- | ---- |
| 怀孕4个月以内 | 15天，特殊情况凭医务部门意见可适当延长，最多不超过30天 |
| 怀孕4个月（含）以上 | 42天 |
- 产假工资：产假工资即生育津贴，按女职工本人生育当月缴费基数除以30再乘以产假天数计算。生育津贴由社会保险经办机构拨付至用人单位，用人单位按实际金额发放给员工。

#### 4. 哺乳假
- 前提条件：遵守国家计划生育政策（已婚、计划内生育），且有不满一周岁婴儿的女职工。
- 假期时长：每天两次，每次30分钟（含路途时间，可合并为每天一小时，不可累积存休），直至婴儿一周岁；多胞胎生育的，每多哺乳一名婴儿，每次增加30分钟；哺乳期满时值盛夏，可适当延长1-2个月；婴儿身体特别虚弱的，凭医务部门证明可酌情延长。

#### 5. 丧假
- 适用情形：员工三代以内直系亲属及配偶父母死亡。
- 假期时长：丧假3天；需赴外地料理丧事的，根据路程远近另给予路程假。
- 休假要求：丧假包含休息日和法定休假日，需一次休完；丧假为带薪假。

#### 6. 年休假
- 适用条件：入司后连续工作满一年以上的员工。
- 假期时长：
| 工作年限 | 年休假天数 |
| ---- | ---- |
| 1年≤工作年限<10年 | 5天 |
| 10年≤工作年限<20年 | 10天 |
| 工作年限≥20年 | 15天 |
- 特殊规定：开始享受年休假后，当年在职时间不足完整年度的，按实际在岗月份折算休假天数；国家法定休假日、休息日、婚假、丧假、产假及工伤停工留薪期间不计入年休假；年休假可集中安排、分段安排或冲抵病事假，不跨年度安排；确因工作需要未使用的假期，经部门负责人签字、人事部核准，可延至次年度第一季度使用，到期未使用的不予补休；年休假为带薪假。
- 不享受年休假情形：累计工作满1年不满10年，请病假累计2个月以上；累计工作满10年不满20年，请病假累计3个月以上；累计工作满20年以上，请病假累计4个月以上。

### 六、加班和调休制度
#### 1. 加班前提
公司提倡通过改进工作方式、提高工作效率完成工作，原则上不鼓励加班。仅在以下情况之一时，部门可组织员工加班：
- 公司原因需安排加班的。
- 为完成公司下达的紧急任务的。

#### 2. 加班规定
- 加班起止时间以打卡记录为准。
- 加班需一次一申请，禁止月累计填写申请单。
- 工作日加班时间一般为1小时，经协商且保证员工身体健康的前提下，每天最多不超过3小时，每月累计不超过36小时。
- 加班申请提交时间：工作日应急加班于当天18:00前；周末加班于加班前最后一个星期五18:00前；法定节假日加班于加班前一周。特殊情况无法按时提交的，由部门主管电话通知人事部，于正常上班后第一个工作日18:00前补交《加班申请单》。

#### 3. 调休规定
- 工作日零星加班、周末加班及特殊紧急任务统一加班，原则上以调休方式补偿；调休最小单位为半天，不足半天的可累计后调休。
- 调休需提前1-2个工作日填写《调休申请单》，经部门主管审批后交人力资源部备案。
- 调休时间最长原则上不超过一周，具体由部门主管安排。
- 请病假、事假可使用加班时长冲抵。
- 加班时长需在一年内调休完毕，每年末结清，不允许跨年调休。

### 七、薪酬管理制度
#### 1. 薪酬保密
员工不得打探其他员工薪酬，也不得向他人透露本人薪酬；非因本人原因获知的其他员工薪酬信息，需履行保密义务。

#### 2. 发薪日期
每月15日发放工资，逢节假日则顺延或提前发放。

#### 3. 月工资构成
月工资=基本工资+绩效工资+餐费补贴+全勤奖（根据出勤情况确定）+其他补贴-个人相关扣款+奖金（根据公司整体效益和个人综合考核确定）+课时费（仅适用于培训部，按讲师级别确定标准，参照《培训部讲师管理制度》）。

### 八、公司福利
#### 1. 办公与生活福利
- 提供优雅舒适的办公环境。
- 每日25元高标准午餐补贴。
- 每月集中组织生日庆祝活动。
- 茶水间提供茶点、零食、咖啡、红茶、绿茶、红糖姜水、可可、水果沙拉、冰糖雪梨汤等。
- 每日15分钟人性化茶歇时间。
- 不定期组织品酒活动。
- 配备健康安全的电水壶、高端咖啡机。
- 提供免洗洗手液、擦手纸、面巾抽纸。
- 发放防雾霾3.0PM口罩。
- 配备咖啡杯、马克杯、品酒酒杯。
- 提供温暖盖毯。

#### 2. 保障类福利
- 缴纳养老、工伤、失业、生育、医疗五险。
- 购买意外伤害险。
- 提供入职体检及年度健康体检。
- 节假日发放礼品。
- 不定期组织团体活动（如春游、聚餐、K歌等）。
- 女员工享受每月1天无条件休假。

## 第二章 行政管理制度
### 一、行为规范
1. 工作时间严禁玩游戏、看电影、聊私事、浏览与工作无关的网页。
2. 爱护公物，节约开支，杜绝浪费。
3. 服从领导安排，关心下属同事，团结互助。
4. 遵纪守法，忠于职守，克己奉公。
5. 严禁蓄意破坏公司网络或窃取公司受保护的网络信息。

### 二、规章制度
#### 1. 仪容仪表与着装
上班时间需保持得体、大方、整洁的仪容仪表；不得穿拖鞋、背心上岗。

#### 2. 电话使用规范
使用电话需注意礼貌用语，控制通话时间；不得使用办公室电话拨打私人电话；公司总机电话由工位人员自行接听，同事不在岗时，其他同事需协助接听、代办并做好记录。

#### 3. 通讯工具与软件使用
所有员工需保持手机、微信、邮件24小时开通，下载并使用“明道”软件客户端；公司事务邮件发送即视为已传达。

#### 4. 电脑使用管理
遵循“谁使用谁负责”原则；长时间外出或下班时，需关闭电脑主机及显示器；办公期间节约网络资源，不影响公司网络服务速度。

#### 5. 打印机使用规范
行政部门每月月底对打印机进行定期保养检查；不得在打印机、传真机上放置重物、液体；打印材料需先定稿，尽量双面打印/复印，节约用纸；爱护公共办公设备。

#### 6. 办公用品管理
- 行政部统一负责办公用品的采购、登记与管理。
- 员工领用办公用品需填写《办公用品领用登记表》，向行政部申请。
- 需购置办公用品的，需经领导审批后，由行政部统一采购。
- 行政部每月月底统计各部门办公用品领用数量，报总经理助理处。
- 办公用品使用需遵循节约、爱护原则，不得随意破坏、丢弃。

#### 7. 印章管理
- 公司印章统一由总经理管理。
- 文件需盖公司印章的，需向行政部申请并填写《印章使用登记表》；不得在空白纸、空白表格、空白合同等空白载体上盖章。
- 所有盖有公司印章的文档，需交总经理办公室统一保管。

#### 8. 办公区域卫生管理
- 员工需保持个人卫生及办公区域干净整洁，随手捡拾纸屑垃圾。
- 中午就餐后，自行将餐后垃圾装入塑料袋，倾倒至办公室外垃圾房，避免产生异味。
- 禁止在地毯上倾倒粉状物体、液体。
- 上班前或下班前，整理好个人办公台面。
- 保洁阿姨每日早晨、下午进行卫生打扫。
- 最后离开办公室的同事，需检查并关闭门、窗、电、水。

#### 9. 水电管理
- 行政部负责公司用电的电量管理及充值，确保正常供电。
- 不得私自毁坏水表、电表。

### 三、发文制度
#### 1. 适用文件范围
本制度所指文件，是以公司或总裁办名义发布、面向全体员工的通知、通报、规章制度等下行公文。部门内部发布的规章制度、通知、通报等，需在行政部备案，不受本制度约束。

#### 2. 管理机构与职责
- 行政部为公文处理管理机构，主管公司各类公文处理工作。
- 各部门内勤（无内勤的部门指定专人）负责本部门对内对外公文处理工作。

#### 3. 文件发布形式
包括纸质件和电子件两种形式。

#### 4. 文件发布流程
- 通知、通报等下行公文：起草校对完毕→填写《XXXX公司发文签批表》→部门负责人审核签字→行政部审核签字→总裁办签发→按签发意见发布→文件登记、编号。
- 规章制度等重要文件：行政部提交公司经营班子成员审核征求意见→根据意见修正→提交总裁办讨论通过→填写《XXXX公司发文签批表》→总裁办签批→发布。

#### 5. 文件流转与管理
| 权责部门 | 输入 | 流程 | 输出 |
| ---- | ---- | ---- | ---- |
| 主办职能部门 | 无 | 文件起草/修订 | 文件 |

### 四、员工意识
#### 1. 团队意识
具备团队合作精神，积极参与团队活动，配合完成工作任务，共同营造和谐、高效的团队氛围。

#### 2. 诚信意识
工作中秉持诚实守信原则，不弄虚作假、不隐瞒事实，保证工作信息的真实性与可靠性。

#### 3. 节约意识
工作和生活中节约公司资源（包括办公用品、水电等），杜绝浪费，降低公司运营成本。

#### 4. 保密意识
严格遵守公司保密规定，对公司商业秘密、技术秘密、客户信息等重要信息履行保密义务，不向外部泄露，不随意在内部传播。
